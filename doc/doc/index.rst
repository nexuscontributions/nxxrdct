==============
|nxxrdct_icon|
==============

The goal of this project is to provide a powerful, user-friendly API for creating and editing the `nxxrdct <https://manual.nexusformat.org/classes/applications/nxxrdct.html>`_ application definition.


.. note::

   NeXus uses the `McStas coordinate system <https://www.mcstas.org/>`_. Make sure your nxxrdct files follow the same conventions. See the :ref:`coordinate_system` section.
